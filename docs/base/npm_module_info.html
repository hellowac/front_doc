查看模块信息

我们可以查看 全局 或 项目 下已安装的各模块之间的依赖关系图，下面是语法和结果：

npm list/ls/la/ll [-g]

结果:

  │     Rational approximation with bounded denominator

模块安装路径

我们可以查看 全局 或 项目 中模块安装的路径：

npm root [-g]

npm list -g --depth 0
